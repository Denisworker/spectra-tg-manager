## О продукте

**Spectra** — профессиональный инструмент для управления множеством Telegram аккаунтов с единой панели управления. Разработан для фармеров, маркетологов и всех кто работает с несколькими аккаунтами одновременно.

---

## Функционал

### Управление аккаунтами
- Импорт tdata аккаунтов — автоматическое определение при добавлении в директорию
- Одновременный запуск нескольких окон Telegram Desktop в изолированных профилях
- Проверка валидности аккаунтов через Telethon API
- Автоподтягивание данных: имя, username, номер телефона

### Сессии
- Конвертация tdata → .session Telethon для автоматизации через API
- Копирование StringSession в один клик

### Прокси
- Назначение индивидуального SOCKS5 прокси на каждый аккаунт
- Массовая загрузка и автоматическое распределение пула прокси
- Встроенная проверка прокси с отображением латентности

### Антибан
- Смена device fingerprint — каждому аккаунту назначается уникальное устройство (iPhone/Android)
- Изолированные runtime-копии tdata — оригинальные файлы не затрагиваются
